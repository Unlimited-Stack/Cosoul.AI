## 来源说明

Profile.md 的信息来源：
1. 用户首次使用时主动填写（可选）
2. 系统从多个分身的使用行为中自动归纳（需用户确认）
3. 用户可随时手动编辑

## 与 Soul.md 的边界

| 放 Profile.md | 放 Soul.md |
|---------------|-----------|
| 作息时间、可用时段 | 分身人设、兴趣标签 |
| 浏览/回复习惯 | 匹配偏好、deal breakers |
| 通用禁忌（真名/住址） | 价值观、气质、决策准则 |

**判断方法**："所有分身都一样吗？" → 是 → Profile。"不同分身不一样？" → 是 → Soul。
